배열(Array)

배열이란 모든 자료형의 데이터를 저장할 수 있는 '변수'의 묶음이다. 자바에서는 컬랙션에 List와 거의 흡사하다.

배열의 선언

new를 이용한 배열 선언과 리터럴로 선언

array객체의 메소드

indexOf() : 배열에서 요소가 위치한 인덱스 리턴

concat() : 두 개 이상의 배열을 결합. 배열 합치기

join() : 배열을 문자열로 반환

sort() : 배열을 정렬

push() : 배열의 맨 뒤에 요소 추가, pop() : 배열의 맨 뒤에 있는 요소를 제거하고 꺼내옴.(튀어나온다.)

shift() : 배열의 맨 앞에 요소 제거, unshift() : 배열의 맨 앞 요소 추가

slice() : 배열의 요소 선택 선택 잘라내기, splice() : 배열의 인덱스 위치에 있는 요소 제거 및 추가
